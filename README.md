battle-board
============

![Accueil Battle Board](http://www.dreamcat.fr/github/battle-board-home.png)

- Installer MongoDB (2.4.4)
- Installer Node.js (0.8.18)
- npm install
- npm install -g supervisor
- launchDev.cmd / launch.sh
- Se connecter en admin/battle
- Créer des équipes
- Créer des joueurs
- Créer un battle
- Gérer les scores
